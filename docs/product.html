How CartGuard flags missing evidence for AI-generated claims.

Rules are clear. Missing items are clear. No hidden logic.

The Problem

  • AI claims often ship without the right supporting evidence.
  • Manual review is slow and hard to scale.
  • Many checks happen too late, right before launch.

The CartGuard Approach

  • Store claims in a strict, typed format.
  • Define required evidence/documents by policy.
  • Run checks in CI and return machine-readable missing-item results.

/packages/spec   -> claim + rule shapes

/packages/cli    -> CLI commands for CI

Important: CartGuard flags missing evidence. It does not make legal determinations.
